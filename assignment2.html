Aira - Anonymous Chatting Application

Overview

Aira is a unique chatting application that allows users to communicate without the need for phone numbers or email registrations. Designed for privacy-conscious users, Aira ensures seamless and anonymous interactions while maintaining a simple and user-friendly interface.

Objective

The goal of Aira is to provide a secure and hassle-free communication platform where users can chat instantly without sharing personal details. It focuses on privacy, ease of use, and accessibility for all users.

Technologies Used

  • MIT App Inventor: A visual programming platform for designing and developing mobile applications.
  • Cloud Database: Used for real-time message storage and retrieval.
  • User Authentication: Custom logic to enable anonymous user access without requiring emails or phone numbers.

Features

  • No phone number or email required for login.
  • Instant messaging with real-time updates.
  • Secure and anonymous communication.
  • Lightweight and efficient for fast performance.
  • Simple and user-friendly interface.

Role & Contributions

As the developer of Aira, I was responsible for:

  • Designing and developing the entire application using MIT App Inventor.
  • Implementing a secure and anonymous login system.
  • Setting up cloud-based data storage for real-time chat functionality.
  • Creating an intuitive and visually appealing user interface.
  • Testing and optimizing the app for better performance.
