Non-Invasive Glucometer - PCB Design using KiCad

Introduction:

This project is based on a non-invasive glucose monitoring device using bio-impedance analysis. The PCB was designed to interface an impedance analyzer with signal conditioning components like AD620, microcontroller, and BLE module.

Design Highlights:

  • Used KiCad for schematic and PCB layout
  • Included analog front-end: AD620 instrumentation amplifier
  • Controlled signal path for impedance measurement with electrode interface
  • Optimized layout for analog precision and minimal noise
  • Prepared Gerber files for fabrication

Conclusion:

This project enhanced my skills in analog circuit design, proper routing for low-noise signals, and multi-layer PCB planning. I also learned the complete pipeline from schematic to Gerber export in KiCad.

555 Timer Based Projects - Milled PCB Designs

Overview:

Alongside KiCad, I explored practical PCB fabrication using PCB milling. I designed and milled compact single-sided PCBs for 555-based circuits like LED blinkers, sound generators, and delay timers.

Design Features:

  • Used 555 timer in astable and monostable configurations
  • Designed simple PCB traces and pads in KiCad
  • Converted layout to SVG and G-code for CNC milling
  • Milled on copper-clad boards using a CNC machine
  • Manually soldered SMD and through-hole components
